## ğŸ® How to Play

1. Click the "Roll" button to roll all dice
2. Click on individual dice to "hold" them (they'll stay locked on subsequent rolls)
3. Keep rolling the unheld dice until all dice show the same number
4. Try to complete the game in as few rolls as possible!
5. Click "New Game" to start over
